# Proyecto_Invent
Creación de un inventario, utilizando POO.

El sistema debera permitir registrar y gestionar productos y movimientos dentro del inventario
Cada módulo estara separado, Model guardara los datos, View mostrara la información y Controller
hará la conección con todo a lo interno. Los integrantes trabajarán en su parte usando ramas y luego unificandolas a la rama main.
